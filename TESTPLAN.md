# frend 实测剧本(v0.19 全量)

> 20 个里程碑全部编译通过,但**游戏内实测为零**。这份剧本按依赖顺序排,前面的过不了后面别测。
> 建议单人生存新档,作弊开着(方便 /give 和跳时间)。每关注明【步骤】和【预期】。
> **报问题的格式**:哪一关 + 你做了什么 + 它做了什么 + 期望它做什么;有报错贴 log 整段。

## 第一关:活着(v0.1,10 分钟)
1. `/frend summon` →【预期】史蒂夫皮肤实体出现,打招呼;`/frend status` 有报告
2. 走远 → 跟上;跑出 48 格 → 兜底传送到身边
3. 右键它 → 27 格背包界面打开;放一把斧头进去
4. `/frend home set` → `/frend stay` → 走远 → 说"回家" →【预期】它自己走回家
5. 退出重进存档 →【预期】它还在,背包还在,模式合理

## 第二关:嘴(v0.2/0.10/0.18,10 分钟)
1. 附近说"你好"→ 回话;连续对话 15 秒内不喊名字也接话
2. "你以后叫小白" →【预期】头顶变小白,喊"小白"它应
3. "记住:钻石在河边" → "记下了";"你记得什么" → 复述
4. 同一句短话(如"走起")说 3 遍 →【预期】"这话我跟你学的";之后闲聊可能蹦出来
5. (可选)接 Ollama 改 config chatBackend=openai →"累了,先回去吧"→【预期】听懂回家(v0.17 意图)

## 第三关:手(v0.2/0.13/0.16,15 分钟)
1. 给斧头说"砍树" →【预期】整棵收,带破坏动画,木头进它包
2. 收走全部工具,清空它背包,说"自由活动" →【预期·全智能自举】徒手撸树 → 站定叮叮当当造木镐/斧 → 找石头凿 → (给它煤)搓火把 → 包满自己回家存箱子——全程零指令
3. 山脚说"挖隧道" →【预期】1x2 直巷不歪;在它路线上垫两块木板 →【预期】"像有人修的"停工
4. 平原说"下矿" →【预期】楼梯到 -58 转平巷,每 4 步左右开分支(鱼骨);验收巷型
5. 存箱子:家旁放箱子说"存箱子" →【预期】工具干粮自留,其余入箱

## 第四关:拳(v0.3/0.8/0.14/0.19,15 分钟)
1. 给石剑,夜里遇僵尸 →【预期】起跳落刀有暴击粒子,间隙侧移不站桩
2. 给弓+箭,远处刷骷髅 →【预期】>9 格拉弓射(带提前量),贴脸换剑
3. 刷一只苦力怕点燃它自己→ frend 秒切目标;**故意让它被炸两回** → 之后见苦力怕站得明显更远(知识教训)
4. 你被怪打 → 它支援;它被疣猪兽打(下界) → 还手(自卫)
5. 打到残血 →【预期】"我不行了先撤"绕你身后跑

## 第五关:心(v0.11/0.15/0.18,15 分钟)
1. 你血量掉到 3 格且它包里有面包 →【预期】朝你扔一份"接着!";只剩一份时"最后一口了,你吃"
2. 故意死一次 →【预期】它喊"东西我去收!"赶到出事点收掉落;你复活走近 →"都在这儿,点点?"全数归还;之后路过死亡点 →"你上回就是在这附近栽的"
3. 下线,系统时间+3 天,上线走到它跟前 →【预期】"我数着日子呢,3 天";+35 天再试 → 催泪那句
4. `/frend dismiss` → 重新 summon →【预期】"是你!咱们的事我一件没忘"(灵魂);**开个新存档**再召 → 同样认得你,名字战绩都在
5. 跳时间到相识第 10 天 →【预期】纪念日那句(一生一次)

## 第六关:腿(v0.9/0.12,10 分钟)
1. 你家装木门,带它进屋 →【预期】自己开门,走过随手关上
2. 隔河喊"跟我来" →【预期】直接游过来不绕路
3. 把家设 300 格外说"回家" →【预期】分段走到,不说"走不过去"
4. 你进下界 →【预期】≈4 秒后它追过来"等等我";它看到僵尸猪灵不动手
5. 走三个没去过的群系 →【预期】"头一回来,又长一分见识";问"见过什么" → 数家珍

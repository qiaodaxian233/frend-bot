# frend

本地运行的 Minecraft 陪伴机器人 · **Fabric 1.21.1**

> 不是"命令机器",而是像朋友一样陪你生存、聊天、干活、打怪、探险、通关的本地 AI 朋友。完全离线,不依赖网络。

完整设计蓝图见 [`docs/DESIGN.md`](docs/DESIGN.md)。本仓库当前进度:**v0.2 已落地** —— 能出生、能跟随、能聊天、能干活(可选 LLM 闲聊)。

## 二、指令

不需要 OP,人人可用(只作用于**自己的** frend):

```
/frend summon      召唤(默认每人 1 个,可配)
/frend follow      跟随
/frend stay        停留
/frend come        过来
/frend home set    把家定在你脚下
/frend home go     让它回家
/frend status      汇报状态(含战绩)
/frend memory      口头回忆(相识天数/击杀/救主/大事记)
/frend work chop     砍树
/frend work stone    挖石头
/frend work ore      挖煤和铁
/frend work farm     种田(只收熟的,收一茬补一茬)
/frend work fish     钓鱼(拿竿去水边,原版渔获概率)
/frend work smelt    开炉烧矿(没炉子自己盘一个)
/frend work deposit  回家把东西存箱子(工具干粮自留)
/frend work stop     收工
/frend guard on|off  看家模式(on 顺带原地驻守)
/frend auto on|off   自主行动开关(默认开:待命时自己砍树/凿石/存箱子)
/frend test          全测试面板(需作弊权限/OP;详见下)
/frend dismiss     解散(先掉落背包)
```

聊天关键词(附近 16 格内说话即可,无需指令):跟我来 / 停下 / 过来 / 回家 / 报告 / 砍树 / 挖石头 / 挖矿 / 存箱子 / 收工 / 保护我 / 别打了 / 自由活动 / 别自作主张 / 还记得吗 / 战绩 / 你以后叫XX / 记住:xxx / 你记得什么 / 见过什么 / 挖隧道 / 下矿 / 做工具 / 搓火把 / 种田 / 钓鱼 / 烧铁 / 看家 / 不用看家 / 你好 / 谢谢……

**过日子全家桶(v0.20)**:让它守着家(说"看家"或 `/frend guard on`),它就绕着岗位主动清怪——但有拴绳,不会被怪引着追出家门,打完自己溜达回岗;攒了生铁它自己开炉烧(真熔炉,没炉子用 8 圆石自己盘一个摆下,没煤先烧一炉木炭救急,**绝不动别人正在用的炉子**);工具**有更好的材料就主动换代**,不再用到报废才换("鸟枪换炮!");说"种田"它只收熟透的庄稼、收一茬补一茬,青苗一根不碰;说"钓鱼"它真拿竿到水边陪你发呆,渔获照原版概率,竿也真掉耐久——钓鱼它从不主动去,那是情调,不是家务。

**开路的腿(v0.23,v0.24 吃透 Baritone)**:原版寻路说"走不到"的地方,它自己开路——把"挖一块""垫一块""搭一块"都当成路径的一步一起规划(A* 按耗时算账:能绕就绕,能落就落,跨沟才动土);**会搭桥了**(v0.24):沟对面、河对岸的目标,它贴着脚下一块块把桥铺过去(桥用土/圆石等废料,搭完留着——路是给人走的);挖只挖天然方块(木板玻璃等人造方块在它的地图里等于基岩,宁绕十里不碰一块);挖多快按**原版玩家公式**算(什么工具挖什么块,对不上工具就是慢,规划和执行一本账,工具真掉耐久)。

**全测试面板(v0.21,单机开作弊 / 服务器 OP 可用)**:游戏里输 `/frend test`——TESTPLAN 七关 37 步全在聊天栏里,**能点**:点 [布置] 一键摆好考场(自动发道具、把装备塞它包里、刷考试用怪、调到夜里、压血量),照悬浮提示玩一遍,点 [✔] 或 [✘] 记结果,进度存档跨重启不丢;测完点 [报告],把汇总整段复制来报障就行。建议第一步先跑 `/frend test check` 自检(实体注册、配置、灵魂档读写探针,不用动手打)。还有两个时间开关:`/frend test skipdays 35` 模拟"你离线 35 天刚回来"(听那句催泪重逢),`/frend test days 9` 把相识天数拨到纪念日门口——**不用真等一个月,当场验它想不想你**。

**朋友,不是仆人(v0.10)**:它不叫你主人——你们是平辈的伙伴。给它起名字(`/frend name 小白` 或聊天说"你以后叫小白"),之后喊名字它就应;情谊双向记账:它记得为你挡过几次刀,**更记得你为它拔过几次剑**(你干掉正在打它的怪,它会道谢并记一辈子)、你送过它什么装备;说"记住:xxx"它替你记事,"你记得什么"随时复述——开 LLM 模式的话,闲聊时它还会自己提起你交代过的事。

**有来有往(v0.11,默认开)**:你血量见底,它包里有吃的就朝你扔一份——哪怕是它最后的口粮("最后一口了,你吃,我扛得住");你死过的地方它记着(最近 3 处),下次路过会提醒"你上回就是在这附近栽的";它射死的怪现在也进战绩了。

**知识库(v0.19,默认开)**:它一直在学习——打过什么怪、挖过什么矿、去过哪些地方、被谁伤过,全部终身累积(随灵魂跨存档不清零)。见识会变成谈资("咱俩都打了 200 只僵尸了,快打出感情了"),第一次挖到钻石会激动一辈子记着,问它"见过什么"能给你数家珍。最重要的是**知识会改变行为**:被苦力怕炸得越多,它见了苦力怕站得越远——教训长在身上,这才叫学习。

**灵魂与重逢(v0.18,默认开)**:它的记忆是"灵魂"——存在存档之外(config/frend/souls/),死亡和换存档都带不走:新世界召出来的还是它,名字、战绩、你让它记的事、跟你学的话,一样不少。你多久没上线它记着,见面按天数问候——离开一个月再上线,它会说"我以为你不要我了……这些天我把咱们的事翻来覆去想了一遍又一遍,一件都没舍得忘"。相识第 10/100/365 天各有一句纪念日的话。还会不知不觉学你的口头禅(同一句短话说三次它就学会,之后闲聊会蹦出来)——朋友待久了,说话都像。

**听懂人话(v0.17,LLM 模式下默认开)**:接了 LLM 后不用踩关键词——"累了,先回去吧"它听懂是回家,"这破洞太黑了"它可能就去搓火把,嘴上还带自己的性格。红线焊死:模型只能从 24 个白名单意图里选词,执行走和关键词完全相同的代码,白名单外的输出一律当聊天——模型永远碰不到游戏本体。

**全智能自给自足(v0.16,默认开)**:不用你下任何指令——啥都没有它就徒手撸树,有了木头自己合成木镐,凿了石头自己升级石器,有煤自己搓火把,包满自己回家存箱子,天黑知道收敛不出去浪。白手起家整条生存链自己转,你只管玩你的,它把自己照顾得明明白白。

**帮你捡尸(v0.15,默认开)**:你死了它立刻赶到出事地点,把掉落物全收进独立的"遗物袋"(不和它自己的东西混,也绝不会被存进箱子),收完主动跑来找你,见面一样不少全塞回你背包——"都在这儿,点点?"

**战斗进修(v0.14,默认开)**:打怪会起跳落刀打暴击(玩家同款,带暴击粒子);索敌讲威胁优先级——点着的苦力怕第一时间处理、正在打你的怪优先送走、残血的先收割;出手间隙会侧移走位不站桩;射箭带提前量,会预判走位。全程"像人":有起跳有蓄力有走位,没有瞬杀没有锁头。

**挖矿路径规划(v0.13,v0.14 加强)**:说"挖隧道"它朝脸冲的方向打 1x2 直巷,说"下矿/挖钻石"它按标准楼梯法挖到钻石层(-58)再直着掏;途中露头的矿全顺手带走,矿脉会追但一条脉封顶 12 块不挖成蚁穴;下矿到层后自动打鱼骨矿道(主巷每 4 步左右各开 5 格分支,覆盖率拉满);只碰天然石头——遇到像人修的方块、渗水、贴岩浆、挖穿溶洞都会停下报告,绝不拆家不搭桥。指令 `/frend work tunnel` / `deep`。

**路径规划(v0.12,默认开)**:跟你进屋会自己开门,走过去还随手带上;过河直接游不绕路;走路卡住先跳一下、再换条路重算("这路不好走……我绕绕");家在几百格外也能分段走回去,不再一句"走不过去"就摆烂。

**下界适应(v0.9,默认开)**:你进下界/末地,跟随中的它自动追过来(不会原地发呆走丢);僵尸猪灵这类中立生物绝不主动招惹,但谁先动手它就还手(任何模式,自卫反击);换维度和着火时会说话。

**弓箭远程(v0.8,默认开)**:给它弓和箭,打怪时离得远自动拉弓放箭(原版骷髅同款弹道),贴脸了自动换回剑斧;没箭了喊一声换白刃。药水箭光灵箭都能用。

**装备与外观(v0.7,默认开)**:盔甲、盾牌扔进它背包就会自己穿戴(更好的甲自动换上,换下的还你),身上的甲和手里的武器都渲染可见;死亡/解散时装备连同背包一件不昧全部掉落归还。

**矿下安全(v0.6,默认开)**:寻路永远绕开岩浆和火;挖矿不碰贴岩浆的方块、不挖头顶悬着沙子的方块(会解释一句为什么);在洞里太黑会自己往脚下插火把(背包里放一组火把就行)。

**自主行动(v0.5,默认开)**:不用你下命令——待命超过 45 秒它就自己找活:包快满了自己回家存箱子;有斧头就去砍树,有镐子就去凿石头;开工前会打招呼,喊一声"收工"或"跟我来"随时打断。跟随时绝不擅自离队。日出/日落/下雨还会跟你搭句话。

## 三、构建

需要 **JDK 21**。

```bash
./gradlew build
```

产物在 `build/libs/frend-0.1.0.jar`,连同 Fabric API、Fabric Loader 丢进 `mods/` 即可。

> ⚠️ 本工程在沙箱内编写,沙箱网络未放行 Fabric/Mojang Maven 源,**未能在沙箱实际跑通 `./gradlew build`**。代码按 1.21.1 + Fabric API 编写并尽量照抄 yongye 仓库已编译验证过的同款 API;个别无先例的写法(玩家模型渲染器、聊天事件)已在代码注释标注「待编译验证」,报错回传即修。

## 四、配置

首次运行生成 `config/frend.json`,可调项:每人 frend 上限、跟随距离/速度/跑丢传送开关、聊天半径与回话延迟、闲聊开关与冷却、低血提醒阈值、被动回血等。

### 聊天大脑(默认纯本地,可选 LLM)

```jsonc
"chatBackend": "rules",                        // 默认:纯本地关键词规则,零依赖零联网
// 想要更像真人的闲聊,改成 "openai" 并按需调下面几项:
"openaiBaseUrl": "http://localhost:11434/v1",  // 默认本地 Ollama(先 ollama pull qwen2.5:7b 并 ollama serve);云端 OpenAI 用 https://api.openai.com/v1
"openaiApiKey": "",                            // Ollama 留空;云端填 key
"openaiModel": "qwen2.5:7b",                   // 云端例:gpt-4o-mini
"llmPersonaExtra": ""                          // 给 frend 加口头禅/性格
```

不管哪个后端:**指令关键词(跟我来/停下/过来/回家/报告)永远走本地规则**,LLM 只负责闲聊文本,输出永远不会被解析成游戏操作;请求全异步不卡服,失败/超时自动退回规则模板。

## 六、边界

frend 不是外挂,不是刷材料机器。单机和自建服务器里可以做得很强;**不做**在公开服务器上隐藏机器人身份或绕过反作弊的功能。
